Information

  • Negative values are handled correctly (e.g. setting event notifications to -5 will send a notification 5 minutes after the event starts).
  • All values will be rounded to the nearest 5 minute interval.
  • No notification can be set more than 30 minutes in advance with the exception of war notifications.
